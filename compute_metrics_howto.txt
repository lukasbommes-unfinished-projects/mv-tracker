1) Set the hyper parameters in `eval.sh` and run
2) Output is generated in the `eval_output` directory in a sub directory which is specififed by the hyper parameters
3) Run the `create_jobfile_for_compute_metrics.py` script to automatically create a jobfile called "compute_metrics_jobfile.txt"
4) Run the command: parallel --jobs 7 < compute_metrics_jobfile.txt to compute metrics in parallel
4) The output will be written in the `mot_metrics.log` file in each sub directory
